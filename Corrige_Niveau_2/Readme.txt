Corrigé de l'ensemble du niveau 2
Pour faire fonctionner le corrigé, importez le fichier niveau2.sql dans votre SGBD.
Ca va créer une base de données "niveau2" contenant les tables utilisées ensuite.
Copiez le dossier "niveau2" dans votre dossier "www" et éditez si besoin le fichier includes/database.php pour modifier les paramètres d'accès à la base de données (si vous utilisez autre chose que root/ mot de passe vide).
Vous pouvez ensuite charger l'url "localhost/niveau2/login.php" dans votre navigateur.

Enjoy!
